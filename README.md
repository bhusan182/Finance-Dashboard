# 📊 Finance Dashboard (Power BI)

## 📌 Overview

This project is a **personal finance dashboard** built using **Power BI** to track, analyze, and visualize income, expenses, savings, and net worth over multiple years (2021–2024).

It provides clear insights into **spending habits**, **saving patterns**, and **financial growth**, helping users make better financial decisions.

---

## 🎯 Objectives

* Track monthly and yearly **income, expenses, and savings**
* Analyze **spending distribution across categories**
* Monitor **saving strategies and allocation**
* Visualize **net worth growth over time**
* Compare **expense vs savings ratio**

---

## 📂 Dataset Description

The dataset includes:

* **Income Sources** (Salary, Secondary Income)
* **Expenses Categories**:

  * House Rent
  * Groceries & Food
  * EMIs
  * Health
  * Leisure
  * Shopping
* **Savings Categories**:

  * Mutual Funds
  * Emergency Fund
  * Fixed Deposits
  * Liquid Cash

---

## 📊 Dashboard Features

### 🔹 1. Expense Analysis

* Breakdown of spending categories
* Major spending areas:

  * House Rent (~40%)
  * Groceries (~24%)
  * EMIs (~21%)

### 🔹 2. Savings Analysis

* Distribution of savings:

  * Mutual Funds (~71%)
  * Emergency Fund (~14%)
  * Fixed Deposits (~13%)

### 🔹 3. Income vs Expense vs Savings

* Year-wise comparison (2021–2024)
* Monthly trend analysis

### 🔹 4. Net Worth Tracking

* Cumulative net worth visualization
* Financial growth over time

### 🔹 5. Detailed Statement Table

* Month-wise breakdown of:

  * Income
  * Expenses
  * Savings
